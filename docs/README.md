# Project Docs (Obsidian vault)

The **repository root** is the Obsidian vault — open it in Obsidian to browse everything (specs,
docs, `.claude/`, code) with working wikilinks. Install the community plugin **Kanban** so
[[board]] renders as a board.

## Map

- [[draft]] — the raw product idea (entry point, Modes 1 & 2).
- [[board]] — the Kanban task board (all tasks across features).
- [[workflow]] — the Spec-Driven + Test-Driven flow, step by step.
- [[task-tracking]] — how tasks are tracked, and how to swap the board for Jira / GitHub Projects.
- [[constitution]] — the project's non-negotiable principles.
- [[documentation]] — the documentation rule (frontmatter + wikilinks structure).
- `adr/` — Architecture Decision Records.

## Conventions

Every documentation note carries YAML frontmatter and links related notes with `[[wikilinks]]`.
See [[documentation]] for the full rule.
