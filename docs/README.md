# Project Docs (Obsidian vault)

The **repository root** is the Obsidian vault — open it in Obsidian to browse everything (specs, docs, `.claude/`, code) with working relative links. Install the community plugin **Kanban** so [board](board.md) renders as a board. This page is the top-level Map of Content; each section below has its own MOC.
